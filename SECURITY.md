# Security Policy

## Supported Versions

The following versions of this project are currently supported with updates to the security.

| Version | Supported          |
| ------- | ------------------ |
| Latest release   | :white_check_mark: |
| Older versions   | :x:                |

## Reporting a Vulnerability
If you encounter a security vulnerability in this project, please report it responsibly.

## How to Report
Do NOT post vulnerabilities in public issues or forums, this allows hackers or exploiters to find vulnerabilities easier.\
Report issues by emailing: ultandoherty04@gmail.com.

## Include the Following Information in the Report
Description of the vulnerability.\
Steps to produce the vulnerability.\
Affected areas or features.\
Any potential Impact.

## Response Expectations
You can expect a response within 1 - 3 days.\
Valid security issues will be acknowledged and addressed.\
If the issue is accepted, a fix will be developed.\
If the issue is declined, an explanation as to why the report does not qualify as a security issue will be presented.
